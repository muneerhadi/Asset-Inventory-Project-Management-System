Inventory Management System

An Inventory Management System built with Laravel and Vue.js to manage products, track stock, and generate reports efficiently.

Features

Add, update, and delete products

Track inventory levels in real-time

Generate reports

Responsive frontend with Vue.js

Backend with Laravel and MySQL

Installation

Clone the repository:

git clone https://github.com/yourusername/inventory-system.git


Navigate to the project folder:

cd inventory-system


Install PHP dependencies:

composer install


Install Node dependencies:

npm install


Set up your .env file and configure the database.

Run migrations:

php artisan migrate


Start the development server:

php artisan serve

Usage

Open the app in your browser at http://localhost:8000

Manage products, check stock, and generate reports easily

Technologies

Backend: Laravel

Frontend: Vue.js

Database: MySQL

Other: Axios, Vue Router
